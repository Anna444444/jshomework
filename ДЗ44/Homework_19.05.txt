Курс: Разработка web-приложений на JavaScript
Тема: Итераторы, генераторы, множества

Задание
=======
Создайте страницу с интерактивной игрой Black Jack для двоих игроков. В качестве второго игрока выступает бот.

1. На странице должно быть:
  * название игры и общий счёт игры по партиям;
  * изображение (аватар) противника;
  * текущая сумма очков противника;
  * поле для открытых карт противника;
  * изображение стола с колодой закрытых карт;
  * поле для открытых карт пользователя;
  * текущая сумма очков пользователя;
  * аватар пользователя;
  * кнопки "Ещё" и "Всё".
Красивый интерфейс и анимация приветствуются.

2. Правила игры. В начале каждой игры берётся новая колода из 52-х карт, и каждому игроку сдаётся по одной карте. Затем игроки по очереди принимают решение - взять ещё одну карту или закончить добор карт.
В раунде побеждает тот игрок, у которого больше сумма очков за карты. Однако если у игрока сумма очков больше 21, это сразу проигрыш. Очки считаются так:
  * карты с номерами - в соответствии с их номером;
  * валет, дама, король - по 10 очков;
  * туз - может считаться как 11 очков или как 1 очко.

3. Для представления колоды карт реализовать класс CardDeck. В классе реализовать метод-генератор для получения следующей карты из колоды.

4. Для представления игроков реализовать следующие классы:
  * класс Player, реализующий абстрактного игрока с пустым методом yourTurn() и списком карт на руках, реализованных с помощью структуры Set;
  * класс UserPlayer, наследник Player, реализующий метод yourTurn() через ожидание действий пользователя (нажатие "Ещё" или "Всё");
  * класс BotPlayer, реализующий метод yourTurn() алгоритмически.

5. Когда игрок проиграл или выиграл партию, через 1-2 секунды должно показываться модальное окно, содержащее текущий счёт по очкам, текст о проигрыше или выигрыше и кнопку "Следующая партия".
